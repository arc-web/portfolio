# Domain Auditor Agent

Part of the ARC forensic audit service. Performs comprehensive domain intelligence gathering for predecessor agency evaluation.

## Features

- WHOIS registration lookup
- DNS record enumeration
- IP geolocation and hosting analysis
- SSL/TLS certificate validation
- Technology stack detection
- Subdomain enumeration and analysis

## Installation

```bash
pip install -r requirements.txt
```
